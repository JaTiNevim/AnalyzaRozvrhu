Detailed Description

Třída určená ke stahování STAG dat a vytvoření "databáze"

Tato třída slouží k vytvoření datové struktury ze STAG dat. Pro stahování se využívá HttpClient. Soubory se ukládají do složky FolderPath_rootFolder. Pokud jsou zde potřebné soubory obsažené, nestahují se nové. Při podivném chování doporučuji celou tuto složku smazat aby se stáhli nové.
